Google Sheets Sync

Syncs all data to organized spreadsheet tabs — Jobs, Inventory, Pricing, Investments, and Monthly Summary with mileage tracking.

1
Create a Google Sheet
Go to sheets.google.com and create a new blank spreadsheet named "Tech Revive Wireless".
2
Open Apps Script
In the sheet click Extensions then Apps Script.
3
Paste the backend script
Delete everything in the editor, paste the code from google-apps-script.js, and save.
4
Deploy as Web App
Deploy then New Deployment then Web App then Who has access: Anyone then Deploy then Allow.
5
Paste your Web App URL below
