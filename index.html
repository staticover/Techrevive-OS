Connect Google Sheets

Follow these 5 steps once and your data syncs across every device, browser, and phone. Takes about 10 minutes.

1
Create a Google Sheet
Go to sheets.google.com, create a new blank spreadsheet, and name it "Tech Revive Wireless".
2
Open Apps Script
In the sheet, click Extensions in the top menu, then click Apps Script. A new tab opens.
3
Paste the backend script
Delete everything in the editor. Paste in the code from the file google-apps-script.js that came with this app. Click the save icon (floppy disk).
4
Deploy as Web App
Click Deploy (top right corner), then New Deployment. Set type to Web App. Under "Who has access" select Anyone. Click Deploy. Google will ask you to authorize — click Allow.
5
Paste your Web App URL here
After deploying, Google gives you a Web App URL. Copy it and paste it below.
